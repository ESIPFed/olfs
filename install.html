Server4 Installation

Ver 1.0.0

First:

Build and install bes and netcdf_handler projects.
Launch the netcdf_handler/opendapserver

Check it out:

svn co http://scm.opendap.org:8090/svn/trunk/Server4

Build it:

ant -f buildfiles/s4.xml server

Install it:

cp build/dist/s4.war $TOMCAT_HOME/webapps rm -rf $TOMCAT_HOME/webapps/s4

Launch it:

$TOMCAT_HOME/bin/startup.sh

Configure it:

vi $TOMCAT_HOME/webapps/s4/WEB-INF/web.xml
(Adjust the values of BackEndServer, BackEndServerPort, and FileSystemPrefix)

Relaunch it:

$TOMCAT_HOME/bin/shutdown.sh $TOMCAT_HOME/bin/startup.sh

 

Go cat. Go.
