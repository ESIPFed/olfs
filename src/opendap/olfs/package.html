This package contains the core code for the OPeNDAP Server-4 servlet, ommonly refered to as the OPeNDAP Lightweight Front-end Server (OLFS). This server acts as a portal to all of OPeNDAP Back End Server (BES) implmentations and is designed to unify server behaviour for various OPeNDAP servers. The OLFS class contains all of the logic for decoding a client request and then making the appropriate response.
