BES

The reference implentation of the BES is based on the C++ implmentation of the OPeNDAP software. To install the BES you will need to download and installt the BES package .

Once installed the BES will require configuration. This achieved by editing a configuration file called "opendap.ini" which must be located in Tell Me Where .

Here is an example opendap.ini file (We should add an extensive comment set to the default file (and this one here) so that we don't really have to go on and on about what's in it.) :

  

   #-----------------------DODS Initialization file------------------------#
   #                                                                       #
   #       National Center for Atmospheric Research (NCAR)                 #
   #       High Altitude Observatory (HAO)                                 #
   #       Key entries for DODS server run time behavior                   #
   #                                                                       #
   #-----------------------------------------------------------------------#
   
OPeNDAP.Help.TXT=/Users/ndp/OPeNDAP/Projects/Server4/swdev/config/dispatch_help.txt
OPeNDAP.Help.HTTP=/Users/ndp/OPeNDAP/Projects/Server4/swdev/config/dispatch_help.html
Fits.Help.TXT=/Users/ndp/OPeNDAP/Projects/Server4/swdev/config/fits.txt
Fits.Help.HTTP=/Users/ndp/OPeNDAP/Projects/Server4/swdev/config/fits.html
DODS.Info.Buffered=no
# If the symbolic name cannot be found in persistence, nice means simply log
   # an error, strict means throw an exception
   DODS.Container.Persistence=strict
# Who is responsable for this server
   DODS.ServerAdministrator=ndp@opendap.org
# Defines size of system global gool
   DODS.Memory.GlobalArea.EmergencyPoolSize=1
   DODS.Memory.GlobalArea.MaximunHeapSize=20
   DODS.Memory.GlobalArea.Verbose=no
   DODS.Memory.GlobalArea.ControlHeap=no
# Where we log the stuff...
   DODS.LogName=/Users/ndp/OPeNDAP/Projects/Server4/swdev/config/opendap.log
   DODS.LogVerbose=yes
# This key defines is the listener handles all client (therefore only one
   # client can be connected at a given time) or if the listener forks other
   # xdods process so multiple clients can be connected in other words the
   # listener just listens... 
   DODS.ProcessManagerMethod=single
# Location of the Unix socket for incoming connections.
   DODS.ServerUnixSocket=/Users/ndp/OPeNDAP/Projects/Server4/swdev/config/opendap.socket
 
# Catalog Configuration
   Catalog.catalog.RootDirectory=/users/ndp/OPeNDAP/Projects/Server4/data
   Catalog.catalog.TypeMatch=nc:.*\.nc;
 
NC.Catalog.Root=/Users/ndp/OPeNDAP/Projects/Server4/data
