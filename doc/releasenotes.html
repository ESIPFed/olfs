Release Notes

Release 1.0 - January 15, 2006

This was an internal release.

Release 1.1 - February 17, 2006

Represents the first prototype release of Server4. It implements the feature set for releases 1.0 and 1.1, along with some a smattering of features due in coming releases. In particular prototype implementations exist for F-23 and F-24 from release 1.2. At the time of this writing the intended feature set is still growing.
