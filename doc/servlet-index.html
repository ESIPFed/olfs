Server4

Greetings,

This is the OPeNDAP Project Server4 software.

Server4 represents a new arrangement of the OPeNDAP server architecture. The old perl/cgi front end request handler that ran inside the Apache web server has been replaced with a Java servlet running inside of a servlet engine (such as Tomcat) . This servlet, called the OPeNDAP Lightweight Front-end Server (OLFS) works in conjunction with a Back End Server (BES) . The BES may (or may not) run on the same system as the OLFS. The OLFS and the BES communicate using TCP/IP sockets. The BES does all of the "heavy lifting" for Server4. Both the OLFS and the BES require installation and configuration before they can be run.
